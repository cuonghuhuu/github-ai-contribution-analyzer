# GitHub AI Contribution Analyzer

A Python-based tool that analyzes GitHub repositories to evaluate contributor activity, calculate contribution metrics, and generate AI-based insights.

---

## 🚀 Features

- Fetch commit data from GitHub API
- Analyze contributions per contributor
- Calculate contribution score
- Rank contributors
- Generate AI-style summaries
- Export results to Markdown report

---

## 🧠 How It Works

1. Fetch commits from a GitHub repository
2. Analyze each contributor:
   - commits
   - additions
   - deletions
   - files changed
3. Calculate a contribution score
4. Generate AI-style summary
5. Export to `report.md`
